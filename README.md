# Plant 100,000 Mangroves — Go Green Website

Website ini merupakan proyek kampanye lingkungan bertema **“Go Green”** dengan fokus pada pelestarian ekosistem mangrove.  
Seluruh konten bertujuan mengedukasi pengunjung tentang pentingnya menanam pohon mangrove serta mengajak masyarakat untuk berdonasi dan ikut beraksi menjaga bumi.

---

## Struktur Proyek

project-folder
1. index.html → File utama (struktur website)
2. style.css → File CSS eksternal (seluruh styling)
3. README.md → Dokumentasi & prompt AI


---

## Desain dan Gaya

Website ini dirancang dengan gaya **modern, elegan, dan natural** menggunakan:
- **Gradasi warna hijau & mint** untuk nuansa alami.
- **Kombinasi font elegan:**  
  - *Playfair Display* (judul, serif – kesan klasik & anggun)  
  - *Poppins* (isi, sans-serif – modern & bersih)
- **Efek hover glowing**, **bayangan lembut (soft shadow)**, dan **tata letak responsif** untuk semua ukuran layar.

Tema warna dan tipografi dipilih agar menggambarkan semangat **pelestarian lingkungan dan keseimbangan alam** 

---

## Fitur dan Section

Website terdiri atas enam bagian utama sesuai ketentuan tugas:

1. **Home (Hero Section)** – pengantar kampanye dan ajakan beraksi.  
2. **About** – penjelasan singkat tentang peran dan pentingnya mangrove.  
3. **Impact** – capaian nyata program dan manfaat bagi lingkungan.  
4. **How It Works** – langkah-langkah kegiatan penanaman.  
5. **Donate** – ajakan berdonasi dengan tombol aksi utama.  
6. **Contact** – form interaktif untuk bergabung sebagai relawan atau bertanya.

## Catatan

- Tidak menggunakan JavaScript, seluruh efek dibuat menggunakan CSS murni.  
- Struktur HTML asli tetap dipertahankan, hanya dilakukan pengaturan ulang elemen dan layout menggunakan class dan selector CSS.  
- Semua gambar bersumber dari **Unsplash** (bebas hak cipta untuk penggunaan edukatif).
